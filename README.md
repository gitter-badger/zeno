# Zeno - Behavioral testing of AI/ML

## Development

### Environment

Please install Conda (or MiniForge) and use VSCode as your editor.

### Install

`conda env create --file environment.yml`
`conda activate zeno`

Install symlinked version for development.

`pip install -e .`

### Deployment
